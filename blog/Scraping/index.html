AO3 Scraper

April 11, 2024

Archive Of Our Own (AO3) does not offer a public API. Therefore, my first step was to find a way to retrieve data from AO3. There are a few unofficial APIs provided by the community. The first one I tried was this one.

Unfortunately, the data didn’t quite meet my requirements.

After some research, I came across this Github page.

This API allowed me to scrape the metadata of a number of fanfiction works from a search link. (Only public fanfiction works)

After the first 2-3 test runs with 100 fanfiction works each, I decided to fetch the first 1000 works, sorted by Kudos, and analyse them in more detail.

Data dimensions The data I was able to retrieve with this API was very promising and provided a good basis for further analyses.

Analyses and findings After analysing this data, I found that I was particularly interested in what content, themes and characters were most popular on AO3. I was fascinated by the many additional tags and wanted to analyse these in more detail. At the same time, I also wanted to find out the differences between the individual fandoms.

Therefore, I decided to analyse the three most present fandoms:

  • Harry Potter - J. K. Rowling
  • Marvel
  • Boku No Hero

to analyse and compare the first 1000 works with the most Kudos according to the same principle.

I therefore had four CSV files, each with 1000 lines and several thousand entries.
